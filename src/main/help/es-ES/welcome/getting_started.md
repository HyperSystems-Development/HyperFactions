# Primeros Pasos

Listo para empezar? Asi se hace:

`/f`
Abre el menu de faccion. Explora facciones, crea
la tuya o revisa invitaciones.

Si te invitaron, revisa la pestana de Invitaciones
y acepta. Si no, busca facciones abiertas o crea
una nueva.

> Una vez dentro, explora el territorio y empieza a reclamar!
