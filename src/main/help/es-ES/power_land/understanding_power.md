# Entender el Poder

El poder permite a tu faccion mantener territorio.
Cada jugador tiene poder personal que se suma al total.

`/f power`
Consulta tu poder y el total de tu faccion.

El poder se regenera estando conectado y disminuye al morir.
> Si los reclamos superan el poder, eres vulnerable!
