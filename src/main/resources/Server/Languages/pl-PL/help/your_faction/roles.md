# Role i rangi

Kazda frakcja ma trzy role w scislej hierarchii. Wyzsze role dziedzicza wszystkie uprawnienia rol nizszych.

---

## Podzial uprawnien

| Akcja | Lider | Oficer | Czlonek |
|--------|--------|---------|--------|
| Budowanie na terytorium | Tak | Tak | Tak |
| Korzystanie z bazy frakcji | Tak | Tak | Tak |
| Czat frakcyjny i sojuszniczy | Tak | Tak | Tak |
| Zapraszanie graczy | Tak | Tak | Nie |
| Wyrzucanie czlonkow | Tak | Tak (tylko Czlonkow) | Nie |
| Zajmowanie / oddawanie terenu | Tak | Tak | Nie |
| Przejmowanie wrogiego terytorium | Tak | Tak | Nie |
| Ustawianie bazy frakcji | Tak | Tak | Nie |
| Usuwanie bazy frakcji | Tak | Tak | Nie |
| Zarzadzanie relacjami (sojusz/wrogosc) | Tak | Tak | Nie |
| Przegladanie logow frakcji | Tak | Tak | Nie |
| Awansowanie do Oficera | Tak | Nie | Nie |
| Degradowanie Oficera | Tak | Nie | Nie |
| Zmiana nazwy frakcji | Tak | Nie | Nie |
| Ustawianie opisu / tagu / koloru | Tak | Nie | Nie |
| Otwieranie / zamykanie frakcji | Tak | Nie | Nie |
| Dostep do ustawien frakcji | Tak | Nie | Nie |
| Przekazywanie przywodztwa | Tak | Nie | Nie |
| Rozwiazywanie frakcji | Tak | Nie | Nie |

>[!NOTE] Oficerowie moga wyrzucac Czlonkow, ale nie moga wyrzucac innych Oficerow. Tylko Lider moze usuwac Oficerow.

---

## Szczegoly rol

- Lider -- Jeden na frakcje. Ma pelna kontrole nad wszystkimi ustawieniami, czlonkami i terytorium. Moze przekazac wlasnosc innemu czlonkowi.
- Oficer -- Zaufani czlonkowie pomagajacy zarzadzac frakcja. Moga zapraszac, wyrzucac czlonkow, zajmowac teren i prowadzic dyplomacje.
- Czlonek -- Domyslna rola po dolaczeniu. Moze budowac na terytorium, korzystac z bazy frakcji i uczestniczyc w czacie frakcyjnym.

>[!TIP] Awansuj swoich najbardziej aktywnych i zaufanych czlonkow na Oficerow, aby pomagali zarzadzac terytorium i rekrutowac nowych graczy.
