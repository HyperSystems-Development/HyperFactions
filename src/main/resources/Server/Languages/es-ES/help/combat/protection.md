# Proteccion de Territorio

El territorio reclamado tiene varias protecciones:

## Proteccion de Bloques
Solo los miembros pueden colocar o romper bloques.

## Proteccion de Contenedores
Cofres, barriles, etc. estan asegurados para los miembros.

## Alertas de Entrada
Recibes notificaciones cuando no-miembros entran en tus reclamos.

> El territorio protege los bloques, no a los jugadores!
