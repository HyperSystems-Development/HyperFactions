# Proteccion de Aparicion

Despues de reaparecer tras la muerte, recibes proteccion
temporal para prevenir el campeo de aparicion.

## Como Funciona

- La proteccion dura **5 segundos** despues de reaparecer
- No puedes recibir dano durante este periodo
- Un indicador visual muestra tu estado de proteccion

## La Proteccion se Rompe

La proteccion de aparicion termina antes si:

- **Atacas** a otro jugador o entidad
- **Te mueves** de tu posicion de aparicion

Esto previene el abuso. No puedes atacar a otros mientras
eres invulnerable. Una vez que realizas cualquier accion,
la proteccion cae y las reglas normales de combate aplican.

---

>[!NOTE] La duracion de la proteccion de aparicion y las condiciones de ruptura son configurables por el servidor. Tu servidor puede usar configuraciones diferentes.

>[!TIP] Usa tu tiempo de proteccion para evaluar la situacion antes de moverte.
