# Formar Alianzas

Las alianzas protegen a ambas facciones del fuego
amigo y disputas territoriales.

`/f ally <faccion>`
Envia una solicitud de alianza. Ambos lados deben aceptar.

Beneficios: sin fuego amigo, visibilidad compartida en el mapa.
> Puede haber un limite en la cantidad de alianzas.
