# Facciones Enemigas

Declarar un enemigo habilita el PvP y la agresion
territorial contra ellos. Accion unilateral.

`/f enemy <faccion>`
Declara enemigo inmediatamente. No requiere acuerdo.

PvP habilitado en el territorio del otro. Se puede
reclamar territorio si se debilitan.

`/f neutral <faccion>`
Restablece la relacion a neutral, finalizando la enemistad.
