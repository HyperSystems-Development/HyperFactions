# Comandos de Economia

Referencia rapida de comandos de economia:

`/f balance`
Ver saldo de la tesoreria.

`/f deposit <cantidad>`
Depositar fondos.

`/f withdraw <cantidad>`
Retirar fondos. (Oficial+)

`/f money transfer <faccion> <cantidad>`
Transferir a otra faccion.

`/f money log [pagina]`
Ver historial de transacciones.
